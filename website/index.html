Ship a tour in 5 minutes

Three steps. No account required.

1

Install the runtime

npm install @trailguide/runtime
2

Create a trail file

// tours/welcome.json
{
  "id": "welcome",
  "title": "Welcome Tour",
  "steps": [
    {
      "target": "#create-btn",
      "title": "Start here",
      "content": "Click to create your first project.",
      "placement": "bottom"
    }
  ]
}
3

Add to your app

import { Trailguide } from '@trailguide/runtime';

// Load your trail and start
const tour = await fetch('/tours/welcome.json');
const trail = await tour.json();

Trailguide.start(trail);

Edit the JSON, reload, see changes. That's it.
